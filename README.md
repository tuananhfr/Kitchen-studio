# Kitchen Studio

**Kitchen Studio** is a modern web-based tool for designing and decorating kitchen interiors.  
It allows users to first create a 2D floor plan (walls, doors, windows), then switch to a 3D environment to place and customize furniture interactively.

---

## Features

- **2D Floor Plan Editor**

  - Draw room layout, walls, doors, and windows
  - Simple drag & drop and snap-to-grid tools
  - Export layout data as JSON for 3D rendering

- **3D Kitchen Decorator**

  - Interactive 3D view using React Three Fiber (Three.js)
  - Drag & drop kitchen furniture (meuble)
  - Change materials, colors, and lighting in real time
